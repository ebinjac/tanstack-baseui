# Teams Directory

The **Teams Directory** provides a full list of every team registered in the Ensemble platform, along with their details and membership counts.

---

## Teams Table

| Column | Description |
|--------|-------------|
| **Team Name** | The official team name on the platform |
| **User Group** | The AD group name for regular user access |
| **Admin Group** | The AD group name for admin access |
| **Members** | Approximate active member count |
| **Created At** | When the team was provisioned |
| **Actions** | View or manage the team |

## Searching & Filtering

Use the search input above the table to filter teams by name or AD group name instantly.
